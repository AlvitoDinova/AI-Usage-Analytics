# User Interface Wireframe Design - AInsight (PHP Native)

This document details the interface layouts and wireframe designs for AInsight using Bootstrap 5 styling with a dominant palette of **White, Blue, and Light Gray**.

---

## 1. Aesthetic Scheme & Theme Settings

- **Primary Colors**:
  - Backgrounds: `#ffffff` (White) and `#f8f9fa` (Light Gray)
  - Branding & Buttons: `#0d6efd` (Bootstrap primary Blue) and `#0a58ca` (Deep Blue)
  - Card outlines/borders: `#dee2e6` (Soft Gray)
  - Text: `#212529` (Dark Charcoal) and `#6c757d` (Muted Gray)
- **Framework**: Bootstrap 5 Responsive Grid System.
- **Layouts**: Standard navigation bar header on public pages; responsive sidebar + main body template on Dashboard pages.
